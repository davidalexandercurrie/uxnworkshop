## anatomy of uxn, varvara

- op codes (32 instructions)
- bytes: 8 bits: 256 values
- stacks 256 bytes
- program memory 64kb / 65536 bytes
- device ports 256 bytes
- varvara (connects devices to device ports, mouse, keyboard etc)
